# A/B Test Analysis — SMS-напоминания

Анализ A/B-теста SMS-напоминаний: расчёт ключевых метрик, оценка uplift и проверка статистической значимости (z-test, chi-square).

## Описание

Проект посвящён анализу A/B-теста нового текста SMS-напоминания.

Цель — оценить, влияет ли новый текст SMS на вероятность оплаты клиента.

---

## Данные

- `customer_id` — идентификатор клиента
- `group_name` — группа эксперимента (`control` / `treatment`)
- `segment` — клиентский сегмент
- `delivered_flg` — факт доставки SMS
- `paid_7d_flg` — факт оплаты в течение 7 дней
- `paid_amount_7d` — сумма оплаты за 7 дней

---

## Постановка задачи

Сравниваются две группы:

- `control` — текущий текст SMS
- `treatment` — новый текст SMS

Целевая метрика — `payment_rate_7d` — доля клиентов, совершивших оплату в течение 7 дней.  
*метрика считается по всем клиентам, а не только по доставленным SMS.*

---

## Подход:

- расчёт ключевых метрик по группам и сегментам
- оценка uplift (разницы между группами)
- проверка статистической значимости

Использованы тесты:

- z-test для сравнения долей (основной)
- chi-square test (дополнительная проверка)

---

## Рассчитанные метрики

Метрики рассчитаны по сегментам и в целом по группам:

- `customers_cnt` —количество уникальных клиентов
- `delivered_rate` — доля доставленных SMS
- `payment_rate_7d` — доля клиентов с оплатой за 7 дней
- `avg_paid_amount_per_customer_7d` — средняя сумма оплаты на клиента

---

## Uplift

Для оценки эффекта нового текста SMS рассчитаны:

- абсолютный uplift — разница `payment_rate_7d` между treatment и control в процентных пунктах
- относительный uplift — рост `payment_rate_7d` относительно control

  
*Основной показатель — абсолютный uplift, так как он напрямую показывает изменение доли оплат.*

---

## Проверка статистической значимости

Для общего `payment_rate_7d` проведена проверка гипотез.


| Гипотеза | Формулировка                                                 |
| -------- | ------------------------------------------------------------ |
| H0       | различий между control и treatment по `payment_rate_7d` нет  |
| H1       | различия между control и treatment по `payment_rate_7d` есть |


---

## Результаты


| Показатель               | Значение   |
| ------------------------ | ---------- |
| payment_rate (control)   | ~21.8%     |
| payment_rate (treatment) | ~25.0%     |
| абсолютный uplift        | +3.17 п.п. |
| p-value (z-test)         | 0.0015     |
| p-value (chi-square)     | 0.0016     |


Оба теста подтверждают статистическую значимость различий.

---

## Вывод

Новый текст SMS показывает положительный эффект: в группе `treatment` доля оплат выше, чем в `control`.  
Новый текст SMS увеличивает вероятность оплаты: рост составляет около +3.17 п.п.   
Эффект подтверждён статистически и может быть использован для принятия продуктового решения.
